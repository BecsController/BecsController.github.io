In the JavaScript language, describe all the use cases for each of:

Parentheses () - Parentheses are used to indicate the argument for a function. So if you assign a function to a block of code that you want to use again, the bit in the parentheses indicates the variable you are using within the function. You also use them around a string ie:
console.log('Hello World!');
Also used to group statements in a loop or if/else statements ie
if: (condition 1) {
statement 1 }
or for (var i=0; i > 0; i -=2)
There's probably other uses that I have missed.

Brackets [] - These brackets are used to indicate an array. You can use them to access an item in an array by its number ie myArray[0], they are used to indicate when there are multiple properties assigned to an object ie:
var cat = {
"enemies": ["water", "dogs"]};

Braces {} - Curly braces are used to indicate a block of code, so you find them everywhere. Anything contained within them is code that's executed together.

Single quotes '' or double quotes "" - Can be used to indicate a string name. Are basically interchangeable providing that you keep to whichever version you have chosen at the beginning and end.

Can be very frustrating problem solving code that turns out you put the wrong quote somewhere... or wrong amount of curly braces.... or put a capital letter somewhere there shouldn't be... sigh.
