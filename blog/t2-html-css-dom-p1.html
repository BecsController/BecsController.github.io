The CSS Zen Garden


Wow, so looking through the CSS Zen Garden was eyeopening. I was blown away with some of the sites, especially the ones with java elements. I cannot wait to get into that.

My Analogy


I have spent a great deal of time trying to explain what I'm learning to my Dad recently lol, so here goes with what I told him.

If you think of a webpage as being like a body, html is the bones. It gives the structure to the body and it is what everything else is built upon. CSS is how your body looks; the muscles and skin etc that are built on top of your bones.

The DOM is the way we section off the body into parts, i.e. the torso, the head, legs, arms etc. If we drew the body and drew lines separating these parts from eachother into squares, it would make it easier to decide what the different parts need to look the way we want them to look. Using squares to separate out our web page in this way is what is known as boxifying design.


What is the box model?


The box model is a system to modify the way the content looks within its box on the page. This is achieved using margins, padding and borders. These parts are nested within one another like a russian doll with the content in the centre, surrounded by padding, which is space between the content and the next part, it's border.

This border which can be displayed in a multitude of different ways is then surrounded by space called the margin. These terms are all pretty familiar to anyone who has spent any amount of time making documents on Microsoft Word or a similar program.
